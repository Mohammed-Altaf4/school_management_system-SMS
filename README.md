🎓 School Management System (SMS) – Flutter App

A School Management System Mobile Application UI built using Flutter.
This project demonstrates how a school management application interface can be designed for managing students, teachers, attendance, and academic activities.

The goal of this project is to create a clean, modern, and user-friendly mobile interface for school administration systems.

📌 Project Overview

The School Management System (SMS) Flutter app provides an intuitive mobile interface that allows school administrators to manage academic activities digitally.

This project focuses mainly on frontend development and UI/UX design using Flutter.
